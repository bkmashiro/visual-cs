Pseudocode

karatsuba(x, y):
  if x < 10 or y < 10:
    return x * y // base case
  n = max(digits(x), digits(y))
  m = ceil(n/2)
  a, b = split(x, m) // x = a*10^m + b
  c, d = split(y, m) // y = c*10^m + d
  z0 = karatsuba(b, d)
  z2 = karatsuba(a, c)
  z1 = karatsuba(a+b, c+d) - z2 - z0
  return z2*10^(2m) + z1*10^m + z0

Key Insight

Karatsuba reduces 4 multiplications to 3 by computing z1 = (a+b)(c+d) - z2 - z0 instead of ac + bd separately.

This gives O(n^1.585) complexity vs O(n^2) for schoolbook multiplication — a significant speedup for large numbers.
