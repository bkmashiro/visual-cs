How It Works

A Bloom filter uses multiple hash functions to map a word to several bits. Insertion sets those bits to 1. Membership testing checks the same positions: if any bit is 0, the word is definitely absent; if all are 1, the word may be present.

  • True positive: the word was inserted and all tested bits are 1.
  • True negative: at least one tested bit is 0, so the word is definitely absent.
  • False positive: all tested bits are 1, but the word was never inserted.
