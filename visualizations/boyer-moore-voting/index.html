Pseudocode

// Phase 1: Find candidate
candidate ← null, count ← 0
for each element x in array:
  if count == 0:
    candidate ← x, count ← 1
  else if x == candidate:
    count++
  else: count--
// Phase 2: Verify
count occurrences of candidate
return count > n/2 ? candidate : none

How It Works

Boyer-Moore Voting finds the majority element (appearing > n/2 times) in O(n) time and O(1) space.

Phase 1: Maintain a candidate and count. When count drops to 0, pick a new candidate. Matching elements increment count, mismatches decrement.

Phase 2: Verify the candidate actually appears > n/2 times (necessary since a majority may not exist).
