Current Operation

Generate items and run the 0/1 knapsack algorithm.

Pseudocode

for i = 1 to n:
  for w = 0 to W:
    if weight[i] > w:
      dp[i][w] = dp[i-1][w]
    else:
      dp[i][w] = max(dp[i-1][w],
        dp[i-1][w-weight[i]] + value[i])
// Backtrack to find selected items

How It Works

0/1 Knapsack maximizes value under a weight constraint. Each item is either taken or left (no fractions).

For each item and capacity, we choose the better of: skip the item (take dp[i-1][w]) or take it (take dp[i-1][w-weight] + value).
