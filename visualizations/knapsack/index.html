How It Works

Recurrence:
If weight[i] > w:
  dp[i][w] = dp[i-1][w]
Else:
  dp[i][w] = max(
    dp[i-1][w], skip
    dp[i-1][w-wt]+val take
  )

Traceback: from dp[n][W], if dp[i][w] ≠ dp[i-1][w] then item i was taken. Move to dp[i-1][w-weight[i]].

Complexity: O(n·W) time and space.
