How it works

A skip list is a layered linked list where each level acts as an "express lane" that skips over many elements, enabling O(log n) average search, insert, and delete operations.

  • Each inserted element is promoted to higher levels based on random coin flips (p = 0.5).
  • Search starts at the top level and moves right or drops down — the yellow path shows this traversal.
  • The HEAD sentinel on the left spans all levels, linking to the first element at each level.
