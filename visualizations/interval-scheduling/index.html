Current Operation

Generate intervals and run the greedy scheduling algorithm.

Pseudocode

sort intervals by end time
selected ← [intervals[0]]
lastEnd ← intervals[0].end
for i = 1 to n-1:
  if intervals[i].start ≥ lastEnd:
    selected.add(intervals[i])
    lastEnd ← intervals[i].end
  else: skip (overlaps)

How It Works

Interval Scheduling Maximization selects the maximum number of non-overlapping intervals.

The greedy strategy: always pick the interval that ends earliest. This provably gives the optimal solution because it leaves the most room for subsequent intervals.
