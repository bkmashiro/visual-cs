Patience Sorting LIS (O(n log n))
Maintain a tails array where tails[k] = smallest possible tail of any increasing subsequence of length k+1.

For each new element x:
• Binary-search tails for leftmost position where tails[pos] ≥ x
• Replace tails[pos] = x, or append if not found (new pile!)
• Track a predecessor pointer for reconstruction

tails always stays sorted → binary search works.
Final tails.length = LIS length.
