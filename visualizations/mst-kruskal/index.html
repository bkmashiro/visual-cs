How It Works

Kruskal sorts all edges by weight, then scans from lightest to heaviest. An edge is accepted only if it connects two different components.

  • Union-Find quickly checks whether the endpoints already belong to the same set.
  • Accepted edges become part of the MST and merge two components.
  • Rejected edges would create a cycle, so they are skipped.
