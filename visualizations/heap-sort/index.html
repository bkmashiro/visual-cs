Heap sort in two phases

The algorithm first turns the array into a max heap, then repeatedly swaps the root with the last heap element and shrinks the heap boundary. This visualizer colors the active heap blue and the finished suffix green.

  • Building Heap restores heap order from the bottom up.
  • Extracting Max moves the largest element to the sorted portion on each pass.
