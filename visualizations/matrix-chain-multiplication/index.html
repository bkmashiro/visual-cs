Pseudocode

for l = 2 to n: // chain length
  for i = 1 to n-l+1:
    j = i + l - 1
    m[i][j] = ∞
    for k = i to j-1:
      q = m[i][k] + m[k+1][j] + p[i-1]*p[k]*p[j]
      if q < m[i][j]: m[i][j] = q, s[i][j] = k

How It Works

Matrix Chain Multiplication finds the optimal way to parenthesize a chain of matrices to minimize scalar multiplications.

The DP table fills diagonally: chain length 2 first, then 3, etc. For each subchain m[i][j], we try every split point k and pick the one giving minimum cost.
