How It Works

Recurrence:
dp[0] = 0
dp[a] = min(dp[a-c]+1) for each coin c ≤ a

Fill amounts 1 → target. For each amount, try every coin that fits and keep the minimum count. If no coin reaches it, dp[a] = ∞.

Traceback: from target, repeatedly follow a → a - chosen_coin until 0.

Complexity: O(amount × coins) time, O(amount) space.
