Pseudocode

T ← T_initial
solution ← random_tour()
while T > T_min:
  neighbor ← swap_two_cities(solution)
  Δ ← cost(neighbor) - cost(solution)
  if Δ < 0 or random() < e^(-Δ/T):
    solution ← neighbor
  T ← T × cooling_rate
return best_solution

How It Works

Simulated Annealing is a probabilistic optimization technique inspired by metallurgical annealing. It explores the solution space by accepting worse solutions with a probability that decreases as the "temperature" cools.

At high temperatures, the algorithm freely accepts worse solutions to escape local minima. As it cools, it becomes increasingly greedy, converging toward the global optimum.
