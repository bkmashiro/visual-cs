How It Works

An LRU cache pairs a hash map with a doubly linked list. The hash map gives O(1) key lookup, while the list keeps keys ordered by recency.

  • get(key) returns the value and moves the node to the head because it just became most recently used.
  • put(key, value) inserts or updates a node, then moves it to the head.
  • If the cache is full, the tail node is evicted because it is the least recently used entry.
