🧠 How KMP Works

KMP avoids redundant comparisons using a failure function precomputed from the pattern.

fail[k] = length of longest proper prefix of pat[0..k] that is also a suffix.

On mismatch at j: jump j → fail[j-1]. The text pointer i never moves backward.

Total: O(n+m) comparisons — linear time!
