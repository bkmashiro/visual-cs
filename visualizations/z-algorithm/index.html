🧠 How Z-Algorithm Works

For a string S, Z[i] = length of longest substring starting at S[i] that is also a prefix of S.

A Z-box [l, r] tracks the rightmost matching segment. For each i:

• If i > r: extend naively from scratch.

• If i ≤ r: copy Z[i−l] as a head-start, then try to extend past r.

For pattern matching: build S = P + "$" + T. Any position in T where Z[i] = |P| is a match!

⚡ Complexity

Z-array buildO(n)
Pattern searchO(n + m)
SpaceO(n + m)
Brute-forceO(n · m)

n = |text|, m = |pattern|. Each char compared at most twice!
