Current Operation

Generate a stream and run reservoir sampling.

Pseudocode

reservoir[0..k-1] ← stream[0..k-1]
for i = k to n-1:
  j ← random(0, i)
  if j < k:
    reservoir[j] ← stream[i]
  else: skip element
return reservoir

How It Works

Reservoir Sampling selects k items uniformly at random from a stream of unknown length n, using only O(k) memory.

Each element at position i has exactly k/i probability of being in the final sample. The first k elements fill the reservoir directly. After that, element i replaces a random reservoir slot with probability k/i.
