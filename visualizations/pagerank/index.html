How PageRank Works

PageRank models the web as a directed graph where each edge represents a hyperlink. A "random surfer" follows links at random, and a node's rank reflects the probability of landing on it after many steps.

  • Initialization — all nodes start with rank 1/N.
  • Iterative update — at each step, node v receives rank from each node u that links to it: PR(v) = (1−d)/N + d · Σ PR(u)/out(u).
  • Damping factor d — probability of following a link (vs. teleporting to a random page). Typically d = 0.85. The (1−d)/N term prevents rank sinks.
  • Dangling nodes — nodes with no outgoing links would absorb rank forever. Their rank is redistributed equally to all nodes each iteration: dangling contribution = Σ PR(dangling) / N.
  • Convergence — iteration stops when the maximum rank change between steps is below ε = 0.0001.
  • Power method — this is equivalent to finding the dominant eigenvector of the Google matrix via repeated multiplication.
