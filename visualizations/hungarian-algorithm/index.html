Algorithm Steps

1. Subtract row minimums
2. Subtract column minimums
3. Cover zeros with min lines
4. If lines = n: find assignment
5. Else: adjust matrix

How It Works

The Hungarian Algorithm solves the assignment problem in O(n^3): assign n workers to n tasks minimizing total cost.

It reduces the cost matrix until enough zeros exist to form a perfect matching. Row/column reductions create zeros; line covering and adjustment ensure optimality.
