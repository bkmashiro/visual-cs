Edge Processing

Kruskal's Pseudocode

sort edges by weight
for each edge (u,v,w) in sorted order:
  if find(u) ≠ find(v):
    union(u, v)
    add edge to MST
  else: skip (would create cycle)
start from vertex 0, add to MST set
while MST set ≠ all vertices:
  find min-weight edge crossing cut
  add that edge and new vertex to MST
