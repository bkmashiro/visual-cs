How The Doubling Trick Works

Start by ranking suffixes using only their first character. After one phase with offset k, each suffix has enough information to compare its first 2k characters.

Sorting pairs (rank[i], rank[i + k]) is enough because both halves were already ranked in the previous phase.

Why The Sentinel Helps

A unique last character such as $ makes every suffix distinct and guarantees one suffix is lexicographically smallest. It also cleanly terminates comparisons.

Where Suffix Arrays Show Up

  • Fast substring search with binary search on sorted suffixes
  • Longest repeated substring through the LCP array
  • Burrows-Wheeler transform and compressed full-text indexes
  • Genome and large-scale text indexing
