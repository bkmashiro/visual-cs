How It Works

Kosaraju's algorithm finds all Strongly Connected Components in O(V+E).

Phase 1 — DFS on the original graph. When a node fully finishes, push it onto a stack.

Phase 2 — Transpose the graph (reverse all edges).

Phase 3 — Pop from stack and run DFS on the transposed graph. Each DFS tree is one SCC.
