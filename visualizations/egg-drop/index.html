How it works

Problem: Given n eggs and a building with k floors, find the minimum number of trials needed to determine the critical floor in the worst case.

Recurrence:
dp[e][f] = min trials with e eggs and f floors

For each floor x to try:
• Egg breaks → check below: dp[e-1][x-1]
• Egg survives → check above: dp[e][f-x]

dp[e][f] = 1 + minx max(break, survive)

Base cases:
• dp[1][f] = f (linear scan)
• dp[e][0] = 0, dp[e][1] = 1

Key Insight: The Tradeoff

With few eggs, you must be conservative — drop from low floors first to avoid wasting all eggs. With many eggs, you can binary search — drop from the middle to halve the search space.

The DP finds the optimal floor x* for each state (eggs, floors) that minimizes the worst case: 1 + max(dp[e-1][x-1], dp[e][f-x]). The gold numbers in each cell show the optimal floor to try first.
