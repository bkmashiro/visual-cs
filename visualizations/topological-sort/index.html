How It Works

Kahn's algorithm repeatedly removes nodes with indegree 0, while DFS records nodes when recursion finishes and reverses that finish order to get a topological ordering.

  • The same DAG is rendered twice so you can compare the two strategies directly.
  • Kahn highlights queue operations and edge removals.
  • DFS highlights active recursion paths and colors nodes by completion order.
