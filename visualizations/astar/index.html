What you're seeing

A* is an informed search algorithm that uses a heuristic (Manhattan distance) to guide the search towards the goal. It combines the actual cost from the start (g) with an estimated cost to the goal (h) to pick the most promising node.

  • Yellow cells are in the open set — candidates to explore next.
  • Blue cells are in the closed set — already fully evaluated.
  • Green cells mark the final shortest path.
  • A* typically explores far fewer cells than Dijkstra or BFS on the same grid.
